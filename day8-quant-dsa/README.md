# Day 8: DSA for Quant Interviews

Master data structures and algorithms commonly asked in quant/HFT interviews.

## Topics Covered
- Array/String Problems (5)
- Math & Probability (5)
- Dynamic Programming (5)
- Graph & Tree Problems (5)
- System Design (5)

## Structure
- `src/quant_dsa.py` - TODO stubs for 25 problems
- `tests/test_quant_dsa.py` - Comprehensive test suite

## Run Tests

cd day8-quant-dsa

## Problems
**Array/String:** moving_average, max_profit_k_transactions, longest_consecutive, merge_intervals, rain_water
**Math/Probability:** random_sampling, reservoir_sampling, shuffle_array, expected_value, monte_carlo_pi
**DP:** optimal_trading, max_profit_cooldown, best_time_with_fee, stock_with_transaction_limit, arbitrage_detection
**Graph/Tree:** shortest_path_dag, topological_sort_tasks, detect_cycle_directed, minimum_spanning_tree, lca_binary_tree
**System Design:** lru_cache, time_series_db, order_book_snapshot, rate_limiter, circular_buffer

## Interview Focus
- Optimal solutions (time/space complexity)
- Edge cases handling
- Clean, readable code
- Explanation of approach
