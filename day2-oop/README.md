# Day 3: Object-Oriented Programming

Master Python OOP concepts through 15 hands-on exercises.

## Topics Covered
- Classes and Objects
- Inheritance and Polymorphism
- Magic Methods (Dunder Methods)
- Properties and Decorators
- Design Patterns (Singleton)

## Structure
- `src/oop.py` - TODO stubs for 15 OOP exercises
- `tests/test_oop.py` - Comprehensive test suite

## Run Tests
```bash
cd day3-oop
pytest tests/ -v
```
